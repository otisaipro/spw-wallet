Add sign-in in 6 lines.

Drop the SDK into any web page. It automatically prefers the extension when installed, and falls back to wallet.spw.network when it isn't.

  • One-time nonce → ECDSA signature (secp256k1)
  • Verify server-side with spw-verify (Node or Python)
  • No OAuth, no email, no password store to breach
index.html
<script src="https://spw.network/connect.js"></script>
<script>
  // nonce from your backend, one-time, 8-128 chars
  const nonce = "a3f19c...9b2e";
  const r = await SPWConnect.signIn({ nonce });
  // r = { address, pubkey, nonce, sig }
  await fetch("/api/wallet/link", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify(r),
  });
</script>
