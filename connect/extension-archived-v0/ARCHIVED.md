# Archived — do not submit

This directory was the v0.x SPW Wallet Connect extension — a thin SDK bridge that
injected `window.spw` on every web page and whose popup only opened
wallet.spw.network in a tab. It was rejected by the Chrome Web Store under the
"Spam and Placement in the Store" policy (reference ID: Yellow Nickel) for being
a launcher-only extension.

The replacement lives in `/opt/sparrow/spw_wallet/extension-v2/` — a complete,
self-custody wallet whose popup contains the full UI. Submit from there.

This directory is kept for reference only.
